Diberikan $a,b,c,d \in \mathbb{R}$. Jika $b,d \neq 0$, tunjukkan $\frac{a}{b} = \frac{c}{d} \iff a \cdot d = b \cdot c$
Bukti:
Diketahui $b,d \neq 0$ maka kita tahu diperbolehkan $\frac{a}{b}=\frac{c}{d}$. dengan hukum komutatif dan asosiatif kita peroleh:
$\frac{a}{b} b \cdot d = \frac{c}{d} b \cdot d \iff ad=bc$  Sebaliknya, diketahui $ad = bc$. Dan karena $b, d \neq 0$ maka $\frac{1}{b}$ dan $\frac{1}{d}$ keduanya ada. Dengan hukum komutatif dan asosiatif dieroleh $ad \frac{1}{b} \frac{1}{d} = bc \frac{1}{b} \frac{1}{d} \iff \frac{a}{b} \frac{c}{d}$.

Dibeirkan $a,b \in \mathbb{R}$ dengan $a,b \neq 0$. Tunjukkan $\frac{1}{ab} = \frac{1}{a} \frac{1}{b}$
Bukti:
Karena $a,b \neq 0$ maka $\frac{1}{a}, \frac{1}{b}$ sedemikian sehingga $\frac{1}{ab}$ ada. Selanjutnya, menggunakan sifat komutatif dan asosiatif diperoleh $(ab) \left( \frac{1}{a} \frac{1}{b} \right) = ab \frac{1}{b} \frac{1}{a} = 1$. Dengan demikian $\frac{1}{ab} = \frac{1}{a} \frac{1}{b}$.

Diberikan $a,b,c,d \in \mathbb{R}$. Jika $b,d \neq 0$, tunjukan $\frac{a}{b} + \frac{c}{d} = \frac{ad+bc}{bd}$ 
Bukti:
Karena $b,d \neq 0$ maka $\frac{1}{a}, \frac{1}{b}$ ada dan tidak nol. Berdasarkan hukum distributif, asosiatif, dan komutatif diproleh: 
$(\frac{a}{b} + \frac{c}{d})(bd) = ad + bc$. Selanjutnya, menurut soal nomor 1. Kita dapat melihat $\frac{a}{b} + \frac{c}{d} = \frac{ad+bc}{bd}$.

Diberikan $a \in \mathbb{R}$. Jika $a.a = a$ tunjukkan a = 0 atau a = 1;
Bukti.
Diketahui $a \cdot a = a$. Dengan menambahkan a pada kedua ruas didapat persaamaan sbg berikut:
$a.a + (-a) = a + (-a) \iff a.a + (-1.a) = 0$
$\iff (a+(-1)) \cdot a \iff a = 1 \text{ atau } a = 0$

Jika $a < b$ tunjukkan -b < -a
Bukti:
Karena a < b, sedemikian sehingga b - a $\in P$ . Akibatnya $-a - (-b) = b - a \in P$
Jadi, -b < -a.

Jika a < b, tunjukkan $a < \frac{a+b}{2} < b$
Bukti:
Karena a < b maka b - a $\in P$. Dengan menggunakan hukum distributif dan dengan mengingat $\frac{1}{2} \in P$ maka diperoleh:
$\frac{a+b}{2} - a = \frac{1}{2} (b+a+(-2)a) = \frac{1}{2}(b-a) \in P$
dan
$b-\frac{a+b}{2}-a = \frac{1}{2}(2b-a+(-b))=\frac{1}{2}(b-a) \in P$
Dengna demikian terbukti $a < \frac{a+b}{2} < b$

Jika $0 < a < 1$ tunjukkan $0 < a^2 < a < 1$
Bukti
Karena $a \neq 0$ maka $a^2 > 0$. Selanjutnya, karena $a > 0$ dan $a - 1 < 0$ maka $a^2 - a = a(a-1) < 0$
Akibatnya $a^2 - a$. Jadi $0 < a^2 < a < 1$

Tunjukkan $\left( \frac{1}{2} (a+b)\right)^2 \leq \frac{1}{2}(a^2 + b^2)$ untuk setiap $a,b \in \mathbb{R}$. Tunjukkan pula kesamaan terjadi jika dan hanya jika a=b
Bukti:
Diperhatikan bahwa $0 \leq (a-b)^2 = a^2 - 2ab + b^2$
sehingga $ab \leq \frac{a^2 + b^2}{2}$. Akibatnya $\left( \frac{1}{2}(a+b) \right)^2 = \frac{1}{4} (a^2+2ab+b^2)$
$\leq \frac{1}{4} (a^2 + a^2 + b^2 + b^2) = \frac{1}{2} (a^2 + b^2)$
Selanjutnya jika a = b maka
$\left( \frac{1}{2} (a+b) \right)^2 = \left( \frac{1}{2} (a+a) \right)^2 = \frac{1}{4} (a^2 + 2a^2 + a^2) = \frac{1}{2}(a^2 + a^2) = \frac{1}{2}(a^2 + b^2)$
Sebaliknya, jika $\left( \frac{1}{2} (a+b)\right)^2 = \frac{1}{2} (a^2 + b^2)$
$\frac{1}{4} (a^2 +2ab+b^2) = \frac{1}{2}(a^2+b^2)$
$\iff \frac{1}{4} (a^2 -2ab +b^2) = 0$
$\iff \frac{1}{4} (a-b) = 0$
$\iff a =b$

Tunjukkan $\frac{1}{-a} = -\frac{1}{a}$ untuk semua $a \in \mathbb{R} - \{0\}$
Bukti:
Berdasarkan soal nomor 3: kita tahu $\frac{1}{-a} + \frac{1}{a} = \frac{1a+1(-a)}{-aa} = 0$
Ini berarti bahwa $\frac{1}{-a} = -\frac{1}{a}$

Tunjukkan tidak ada bilangan asli n sehingga $0<n<1$
Bukti.
Diandaikan ada suatu bilangan $n\in \mathbb{N}$, maka himpunan $E = \{ n \in \mathbb{N}: 0 < n < 1\}$ tidak kosong. Karena $E \subseteq \mathbb{N}$ dan tak kosong, maka menurut contoh 1.7.4 ada suatu $m \in E$ yang merupakan elemen terkecil di E. Karena 0 < m < 1, maka menurut soal nomor 7, $0 < m^2 < m < 1$. Akibatnya, $m^2 \in E$. Hal ini bertentangan dengan m elemen terkecil di E.

Ketaksamaan Bernoulli
Jika x > -1 maka $(1+x)^n \geq 1+nx$ untuk setiap $n \in \mathbb{N}$
Bukti.
Pernyataan akan dibuktikan dengan menggunakan induksi matematika. 
(1) Untuk n = 1, maka $(1+x)^1 \geq 1 + (1)x \to 1+x \leq 1+x$ sehingga pernyataan benar untuk n = 1
(2) Andaikan pernyataan benar untuk n sedemikian sehingga $(1+x)^n \geq 1+nx$ harus dibuktikan untuk (n+1) maka
$(1+x)^{n+1} = (1+x)^n \cdot (1+x)$
$\geq (1+nx)(1+x)$
$= (1+(n+1)+x +nx^2$
$\geq 1+(n+1)x$

Sehingga pernyataan benar untuk n + 1;
