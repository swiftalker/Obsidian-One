Buktikan, jika k dan l masing-masing genap maka k + l genap.
Bukti:
Jika k dan l genap sedemikian sehingga k = 2m, dan l = 2n, dengan $m \in \mathbb{Z}$ karena k + l genap maka k+l = 2m+2n = 2(m+n), karena asumsi genap adalah x = 2y dengan literasi x adalah bilangan genap berapapun dari y. maka jelas bahwa k+l adalah bilangan genap. m+n dapat kita asumsikan sebagai y. Sehingga k+l = 2y.
KoreksI:
Misalkan k dan l genap, maka terdapat bilangan bulat $m, n \in \mathbb{Z}$ sehingga $k=2m$ dan $l=2n$. Maka:
$k+l = 2m + 2n = 2(m+n)$
Karena $m+n \in \mathbb{Z}$ maka $k+l$ merupakan kelipatan 2.
Jadi, k + l adalah bilangan genap.

Buktikan:
a. Jika k genap dan l ganjil maka k + l ganjil.
Bukti:
Misalkan k genap dan l ganjil maka terdapat bilangan bulat $m, n \in \mathbb{Z}$ sehingga k = 2m dan l = 2n + 1. Maka:
$k+l = 2m + 2n + 1 = 2(m+n) + 1$.
Karena $m+n \in \mathbb{Z}$ maka dan teori bilangan ganjil adalah suatu $x = 2y + 1$ dan demikian x dapat dikatakan ganjil.
Jadi, k + l ganjil.

Koreksi:
Misalkan k genap dan l ganjil maka terdapat bilangan bulat $m, n \in \mathbb{Z}$ sehingga k = 2m dan l = 2n + 1. Maka:
$k+l = 2m + 2n + 1 = 2(m+n) + 1$.
Karena $m+n \in \mathbb{Z}$, maka k + l berbentuk 2q + 1 untuk suatu bilangan bulat q. Jadi k + l adalah bilangan ganjil.

b. Jika k genap dan l ganjil maka kl genap.
Bukti.
Misalkan k genap dan l ganjil sedemikian sehingga $m, n \in \mathbb{Z}$ sehingga k = 2m dan l=2n+1. Maka:
$kl = (2m)(2n+1) = 2mn+2m = 2(mn+m)$
Karena $mn+m \in \mathbb{Z}$ maka $kl$ merupakan kelipatan 2.
Jadi, kl adalah bilangan genap.

Buktikan jika n ganjil maka $n^2$ ganjil
Bukti.
Misalkan bilangan n ganjil sehingga terdapat bilangan $k \in \mathbb{Z}$. Maka n = 2k + 1 yang berakibat $n^2$ ganjil. Sehingga:
$n^2 = (2k+1)^2 = (2k+1)(2k+1) = 4k^2 + 2k + 2k + 1 = 4k^2 + 4k + 1 = 2(2k^2+2k) + 1$
Karena $2k^2 + 2k \in \mathbb{Z}$ adalah bilangan ganjil, maka $n^2$ berbentuk $2m+1$ adalah bilangan ganjil.

Buktikan persamaan 2x-c=0 memiliki jawaban tunggal
Bukti.
(1) Terlihat bahwa $2x=c \implies c=2x \implies  x = \frac{c}{2}$. Terlihat bahwa elemen x (dan bahkan c) memenuhi eksistensi/ketunggalan. Dimana ambil $x=\frac{c}{2}$. Maka:
$2x-c=2\left( \frac{c}{2} \right) - c = c - c = 0$. Jadi, persamaan memiliki paling sedikit satu solusi.
(2) Misalkan $x_{1} , x_{2}$ adalah dua solusi dari persamaan tersebut, sehingga:
$2x_{1}-c = 0 \text{ dan } 2x_{2}-c=0 \text{ maka } 2x_{1} = c = 2x_{2}$
Dengan membagi kedua ruas dengan dua diperoleh $x_{1} = x_{2}$. Jadi, solusi persamaan tersebut tunggal.

Buktikan grafik $y=x^2 - 1$ untuk $D(f) = \{ x: x \geq 0 \}$ memotong sumbu -x di satu (tunggal) tempat.
Bukti.
(1) Grafik $y=x^2-1$ memotong sumbu-x di $x^2 - 1 = 0$ dan oleh karena itu kita tahu $x_{1,2} = \{-1, 1\}$. Jadi, persamaan memiliki paling sedikit satu solusi karena kita tahu $x \geq 0 \text{ dan } x^2 = 1$. Sehingga terdapat tepat 1 solusi saja.
(2) Misalkan $x_{1}, x_{2} \geq 0$ adalah dua solusi persamaan tersebut sehingga $y = x_{1}^2 - 1$ dan $y = x_{2}^2 - 1$ maka $x_{1}^2 = 1 = x_{2}^2$. Dengan memberikan -1 pada setiap ruas serta mengkuadratkan masing-masing ruas, dapat diperoleh $x_{1}=x_{2}$ . Jadi, solusi persamaan tersebut tunggal.

Buktikan $\frac{1}{2.1} + \frac{1}{3.2} + \frac{1}{4.3} + \dots + \frac{1}{n(n-1)} = \frac{n-1}{n}$ untuk $n \geq 2$.
Bukti.
(1) Untuk n = 2, kita akan verifikasi dengan $\frac{1}{2(2-1)} = \frac{2-1}{2} \implies \frac{1}{2}=\frac{1}{2}$. Sehingga untuk n = 2 terpenuhi.
(2) Asumsikan n=k dipenuhi sedemikian sehingga, $\frac{1}{2.1} + \frac{1}{3.2} + \frac{1}{4.3} + \dots \frac{1}{k(k-1)} = \frac{k-1}{k}$. Selanjutnya buktikan apaha terpenuhi untuk n=k+1. Untuk n = k + 1:
$\frac{1}{2.1} + \frac{1}{3.2} + \frac{1}{4.3} + \dots + \frac{1}{k(k-1)} + \frac{1}{(k+1)(k)}$ = $\frac{k-1}{k} + \frac{1}{(k+1)k}$
$\frac{(k+1)k(k-1)+k}{k(k+1)k} = \frac{k^3}{k^2(k+1)} = \frac{k}{k+1}$
Sehingga formula berlaku untuk n = k + 1. Karena benar untuk n = 2 dan benar untuk n = k + 1, jika benar untuk n = k, maka rumus berlaku untuk semua $n \geq 2$.

Buktikan (x-y) merupakan faktor dari $(x^n - y^n), n \in \mathbb{N}$.
Bukti.
"$\forall n \in \mathbb{N},  \exists Q_{n} (x,y) \text{ sehingga } x^n - y^n = (x-y)Q_{n}(x,y)$"
(1) Misalkan n = 1 dan pasti $n \geq \mathbb{N}$ sedemikian sehingga $(x^1-y^1 \implies x-y)$ sehingga pernyataan benar untuk n = 1
(2) Asumsikan n = k dipenuhi sedemikian sehingga $x^k - y^k= (x-y)Q_{n}(x,y)$  untuk suatu polinom $Q_{k}$. Asumsikan $xy^k - xy^k = 0$. Akan dibuktikan untuk n = k + 1:
$x^{k+1} - y^{k+1} = x^{k+1} - y^{k+1} + xy^k - xy^k$
$= x(x^k - y^k) + y^k(x-y)$
Dengan asumsi induksi, $x(x^k-y^k) = x(x-y)Q_{n} (x,y)$.
Maka, $x^{k+1} + y^{k+1} = (x-y)(xQ_{{k}}(x,y) + y^k)$.
Karena, $x^{k+1} - y^{k+1}$ dapat ditulis sebagai (x-y) dikali suatu polinom maka $(x-y)|(x^{k+1} - y^{k+1})$. Oleh karena itu, karena basis benar untuk P(k) => P(k+1) maka dengan prinsip induksi matematika (x-y) merupakan faktor dari $(x^n - y^N)$ untuk semua bilangan $n \in \mathbb{N}, n \geq 1$.

Buktikan $1+r+r^2 + \dots + r^n = \frac{1-r^{n+1}}{1-r}$
Bukti.
-
(1) Misalkan n adalah bilangan asli sedemikian sehingga $n \in \mathbb{N}$, dengan asumsi n = 1 maka $r^1 = \frac{1-r^{1+1}}{1-r} \implies r = \frac{1-r^2}{1-r} \implies \frac{(1-r)(1+r)}{1-r} \implies 1+r$ sehingga $r = 1+r$ 
-
(1)  Misalkan n adalah bilangan asli sedemikian sehingga $n \in \mathbb{N}$, dengan syarat $r \neq 1$. Ambil n = 1, sehingga $1+r=\frac{1-r^2}{1-r}$. 
Karena $1-r^2 = (1-r)(1+r)$ maka $\frac{1-r^2}{1-r} = \frac{(1-r)(1+r)}{1+r} = 1+r$. Dengan demikian pernyataan benar untuk n = 1.
(2) Asumsikan n = k berlaku $1+r^k = \frac{1-r^{k+1}}{1-r}$ selanjutnya harus ditunjukkan berlaku untuk n = k + 1: 
$1+r+\dots+r^k+r^{k+1} = \frac{1-r^{k+1}}{1-r} + r^{k+1}$
= $\frac{1-r^{k+1}}{1-r} + \frac{r^{k+1}(1-r)}{1-r} =\frac{1-r^{k+1}+r^{k+1}-r^{k+2}}{1-r} =\frac{1-r^{k+2}}{1-r}$
Dengan demikian ini persis seperti apa yang diekspektasikan pada n = k+1. 

Buktikan $6^n - 1$ habis dibagi 5.
Bukti.
(1) Misalkan n adalah bilangan asli sedemikian sehingga $n \in \mathbb{N}$. Ambil n = 1, sehingga $6^1 - 1 = 5$ jelas habis dibagi 5. Maka pernyataan benar untuk n = 1.
(2) Asumsikan n = k berlaku $6^k - 1 = 5m$ dengan asumsi $m \in \mathbb{Z}$ berarti $6^k - 1$ habis dibagi 5 dan harus ditunjukkan berlaku untuk n = k + 1:
$6^{k+1} - 1 \implies 6^k.6 - 1 \implies 6^k.6 - 6  +5 \implies 6(6^k-1) + 5 \implies 6(5m) + 5 \implies 30m + 5 \implies 5(6m+1)$ sehingga $6^{k+1} - 1$ habis dibagi 5. 

Buktikan 2(n+2) <= (n+2)^2 untuk semua $n \in \mathbb{N}$
Bukti.
(1) Misalkan n adalah bilangan asli sedemikian sehingga $n \in \mathbb{N}$. Ambil n = 1, sehingga $2(1+2) \leq (1+2)^2 \to 2(3) \leq 3^2 \to 6 \leq 9$. Sehingga pernyataan benar untuk n = 1.
(2) Asumsikan n = k berlaku $2(k+2) \leq (k+2)^2$ dan harus ditunjukkan n = k + 1: 
- ruas kiri $2((k+1) + 2) \to 2(k+3) \to 2(k+2) + 2$
- ruas kanan $(n+2)^2 \to ((k+1)+2)^2 \to (k+3)^2$
Kita tahu $(k+2)^2 \to (k+2)(k+2) \to k^2 + 2k + 2k + 4 \to k^2 + 4k + 4$
Juga $(k+3)^2 \to (k+3)(k+3) \to k^2+6k+9$
Sehingga dapat diperoleh $2(k+3) = 2(k+2) + 2 \leq (k+2)^2 + 2 = k^2 + 4k + 6 \leq k^2 + 6k + 9 = (k+3)^2$
Kesimpulan $2(k+3) \leq (k+3)^2$
Sehingga pernyataan n = k + 1 juga benar.

Tunjukkan bahwa $2^n < n!$ untuk semua bilangan asli $n\geq4$.
Bukti.
(1) Untuk n = 4, 
- $2^n = 2^4 = 16$
- n! = 4x3x2x1 = 24
Sehingga pernyataan benar untuk n = 4;
(2) Asumsikan n=k, sehingga $2^k < k!$ dan harus ditunjukkan $n=k+1$:
- Ruas kiri $2^{k+1} = 2^k.2 = 2^k + 2^k$
Dari asumsi ruas kiri kita bisa buat menjadi $2.2^k < 2.k! \to 2^{k+1} < 2k!$
- Ruas kanan $(k+1)! = (k+1) . k . (k-1) \dots 2 . 1 = (k+1) (k.(k-1)\dots2.1 = (k+1)k!$
Karena $k\geq4$ maka $k+1\geq5$ sehingga $k+1> 2$. Maka: 
$2.2^k < 2k! < (k+1)k! \to 2^{k+1} < (k+1)k!$
Dengan demikian pernyataan benar untuk n = k + 1;

DEFINISI FAKTORIAL
Untuk setiap bilangan asli n dengan $n\geq1$
$n! =  n \cdot (n-1) \cdo (n-2) \dots 2 \cdot 1$ 
Kemudian n = k + 1
$(k+1)! = (k+1) \cdot k \cdot (k-1) \dots 2 \cdot 1$
= $(k+1) \cdot (k \cdot (k-1) \dots 2 \cdot 1)$
Sehingga dapat kita asumsikan seperti ini
(k+1)! = (k+1)k!

Tunjukkan bahwa $(n+1)^n < 3n^n$ untuk setiap bilangan asli n.
Bukti.
Sebelum mengerjakan soal dapatlah kita sederhanakan soal ini terlebih dahulu dengan membagi semuanya dengan $n^n$ sehingga:
$\frac{(n+1)^n}{n^n} < 3$
$=(\frac{n+1}{n})^n < 3$
$= (1+\frac{1}{n})^n < 3$

Sehingga dapat dibuktikan lebih lanjut:
(1) Andaikan n = 1 sedemikian sehingga $\left( 1+\frac{1}{1} \right)^1 < 3 \to 2<3$. Dan karenanya pernyataan benar untuk n = 1;
(2) Asumsikan n = k, sehingga $\left( 1+\frac{1}{k} \right) < 3$ dan harus dibuktikan n = k + 1, maka:
Ruas kiri: $\left( 1+\frac{1}{k+1} \right)^{k+1} \to \left( 1+\frac{1}{k+1} \right)^{k} \cdot (1+\frac{1}{k+1})$
Karena di ruaskanan tidak terdapat variabel n apapun maka mudah kita buktikan bahwa $\left( 1+\frac{1}{k+1} \right)^{k} \cdot (1+\frac{1}{k+1}) < 3$ yang berarti pernyataan ini benar untuk n = k + 1.
